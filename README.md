# Blossum Bouquets – Java Web Application 🌸

Blossum Bouquets is a Java-based e-commerce web application developed for online flower bouquet shopping. The application allows users to browse bouquet collections, manage cart items, and place orders through an interactive and user-friendly interface.

---

## 🚀 Features

- User Registration & Login Authentication
- Browse Flower Bouquet Products
- Add to Cart Functionality
- Update & Remove Cart Items
- Order Management
- Dynamic Product Display
- Session Management
- Responsive User Interface

---

## 🛠️ Tech Stack

- Java
- JSP
- Servlets
- MySQL
- HTML
- CSS
- JavaScript
- Apache Tomcat

---

## 🗄️ Database Setup

Run the following SQL command in MySQL:

```sql
CREATE DATABASE blossum_bouquets;
```

Import the provided SQL file into MySQL Workbench.

---

## ▶️ How to Run

1. Install Java JDK
2. Install MySQL Server
3. Install Apache Tomcat
4. Open project in Eclipse Enterprise Edition or IntelliJ Ultimate
5. Add MySQL Connector JAR
6. Configure Tomcat Server
7. Run the project on server
